def check_pangram(text: str) -> bool:

    t = text.lower()    
    for char in "abcdefghijklmnopqrstuvwxyz":
        if char not in t:
            return False
    return True

Here you take lower case letters from standard module, convert both strings into sets and check that all lower case letters are in the given string (one set is a subset of another). 

from string import ascii_lowercase


def check_pangram(text):
    return set(ascii_lowercase).issubset(set(text.lower()))
