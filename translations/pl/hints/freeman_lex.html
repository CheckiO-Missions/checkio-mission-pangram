Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Aby iterować po sekwencji wszystkich liter w tym samym przypadku i sprawdzać, czy każda z nich jest obecna w podanym tekście (w wybranym przypadku). Jeśli dana litera nie występuje, to nie jest to pangram i funkcja może natychmiast zwrócić False. Proszę zwrócić True jeśli wszystkie litery są obecne.

Ok, od czego powinienem zacząć?

Wybierzmy małe litery i przekonwertujmy podane text na tę wielkość za pomocą metody str.lower(). 

t = text.lower()
Co zrobić z tekstem?

Proszę wziąć sekwencję wszystkich małych liter (jak wybrano powyżej), na przykład w postaci prostego łańcucha, i iterować po nim za pomocą pętli dor. 

for char in "abcdefghijklmnopqrstuvwxyz":
    ...
Proszę mi pomóc sprawdzić znak.

Proszę użyć instrukcji if i operatora not in, aby sprawdzić, czy bieżący znak jest obecny w podanym text. 

if char not in t:
    ...

Jak wspomniałem powyżej, jeśli znak nie znajduje się w text, nie jest to pangram i funkcja może natychmiast zwrócić False. 

if char not in t:
    return False
Proszę mi pomóc dokończyć kod.

Jeśli wszystkie litery występują w podanym text co najmniej raz (bez zwracania False ), to text jest pangramem i Pana funkcja musi zwrócić True na końcu.

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

def check_pangram(text: str) -> bool:

    t = text.lower()    
    for char in "abcdefghijklmnopqrstuvwxyz":
        if char not in t:
            return False
    return True

Jasne, proszę łapać!

Tutaj proszę wziąć małe litery ze standardowego modułu, przekonwertować oba ciągi na zbiory i sprawdzić, czy wszystkie małe litery znajdują się w podanym ciągu (jeden zbiór jest podzbiorem drugiego). 

from string import ascii_lowercase


def check_pangram(text):
    return set(ascii_lowercase).issubset(set(text.lower()))

Używając funkcji lambda (anonimowej), proszę ustawić zrozumienie i zliczyć liczbę unikalnych liter w podanym tekście. 

check_pangram = lambda text: len({x for x in text.lower() if x.isalpha()}) == 26
