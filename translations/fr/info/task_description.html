Un pangramme (du grec pan : « tous » et gramma : « lettre ») est une phrase comportant toutes les lettres de l'alphabet. Avec un peu d'ingéniosité, il est facile d'en créer sur le modèle des phrases réflexives : « Jugez que ce texte renferme l'alphabet, dix voyelles, k et w. » (Thérèse Amiel).

Pour cette mission, nous utiliserons l'alphabet Latin (A-Z) qui comporte 26 lettres. Vous recevez en paramètre un texte avec des lettres de l'alphabet latin et des caractères de ponctuation. Vous devez déterminer si cette phrase est un pangramme. La casse des lettres (majuscule ou minuscule) n'a pas d'importance.

Entrée: Un texte sous forme de chaine de caractères.

Résultat: Si la phrase est un pangramme, sous la forme d'un booléen.

Exemple:

{% if is_js %} 
checkPangram("The quick brown fox jumps over the lazy dog.") == true
checkPangram("ABCDEF.") == false
{% else %} 
check_pangram("The quick brown fox jumps over the lazy dog.") == True
check_pangram("ABCDEF.") == False
{% endif %}

Utilisation: Les pangrammes sont utilisés depuis longtemps pour visualiser les caractères, tester du matériel, et développer l'habileté en écriture, en calligraphie ou en dactylographie.

Précondition:

{% if is_js %} All ASCII English Letter(a-z,A-Z) and punctuations such as: 
!"#$%&\'()*+,-./:;<=>?@[\\]^_`{|}~
{% else %}
  • all(ch in (string.punctuation + string.ascii_letters + " ") for ch in text);
  • 0 < len(text).
{% endif %}
